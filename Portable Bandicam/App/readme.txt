=======================================================================
***************** Features of the portable build: ******************
==================================================================================

- The portable version does not litter the system, stores settings in its own folder and can work from any media.
- Automatic registration of the program at first launch, as well as when changing the user or PC.
- The ability to disable the screensaver and run the program in safe mode (-> "Data\Bandicam\Config.ini"). 
- Ability to run the program with UAC enabled without administrator rights (-> "BandicamPortableNonAdmin.exe").

==========================================================================
**************************** Note: ********************************
======================================================================================

- For the portable program to work correctly under an account
with limited rights - do not place it in the "Program Files"
folder or another folder with limited access, or run it as an
administrator.
